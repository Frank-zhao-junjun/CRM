# 简易CRM系统

这是一个基于 Next.js 16 + shadcn/ui 构建的简易客户关系管理系统（CRM），适用于PC端和移动端演示使用。

## 功能特性

### 1. 全局搜索 (P0 功能)
- 导航栏全局搜索框
- 支持搜索：客户、联系人、商机、销售线索、报价单、订单、合同、产品
- 实时搜索建议（300ms 防抖）
- 搜索结果分类展示
- 匹配文字高亮
- 键盘导航支持（↑↓ Enter Esc）
- 点击结果跳转详情页

### 2. 仪表盘
- 客户、联系人、销售机会统计
- 销售漏斗可视化
- **销售数据分析**：阶段转化率图表、赢单预测、管道金额分布
- 最近活动记录
- 最近销售机会列表

### 3. 客户管理
- 客户列表（支持搜索和状态筛选）
- 新建/编辑/删除客户
- 客户详情页（查看关联联系人和销售机会）
- 支持PC端表格和移动端卡片两种展示方式

### 4. 销售机会管理
- 机会列表（支持搜索和阶段筛选）
- 完整的销售漏斗阶段（线索 → qualified → 提案 → 谈判 → 成交/失败）
- 新建/编辑/删除销售机会
- 销售漏斗进度可视化

### 5. 联系人管理
- 联系人列表（支持搜索）
- 新建/编辑/删除联系人
- 联系人详情页
- 支持设置主要联系人

### 6. 报价单管理
- 报价单列表（支持搜索和状态筛选）
- 新建报价单（从商机关联，支持产品明细和折扣计算）
- 报价单详情（编辑、删除、发送、转订单）
- 报价单编辑
- 完整状态流转（草稿 → 已发送 → 已接受/已拒绝/已过期）
- 自动计算小计、折扣、税额和总金额

### 7. 合同管理
- 合同列表（支持搜索和状态筛选）
- 统计卡片（全部/草稿/执行中/已完成/已终止）
- 新建合同（支持从报价单一键创建，自动带入客户和金额）
- 合同详情页（完整合同信息、履约进度跟踪）
- 合同编辑页（仅草稿状态可编辑）
- 履约节点配置（里程碑、预计完成日期）
- 完整状态流转（草稿 → 执行中 → 已完成 → 已终止）
- 履约进度可视化（进度条、节点完成状态）
- 关联管理（客户、商机、报价单）
- 合同条款（标准条款+自定义条款）

### 8. 订单管理
- 订单列表（支持搜索和状态筛选）
- 新建订单（支持从已接受的报价单一键创建）
- 订单详情页（查看订单明细、金额汇总、关联信息）
- 订单编辑（修改产品明细、日期、付款方式等）
- 完整状态流程（草稿 → 已确认 → 待付款 → 已付款 → 已完成 → 已取消）
- 状态进度可视化展示
- 支持订单操作：确认、标记待付款、确认付款、完成、取消
- 自动计算小计、折扣、税额和总金额
- 付款方式管理（银行转账、现金、信用卡等）

### 9. 发票管理
- 发票列表（支持搜索和状态筛选）
- 新建发票（支持从订单创建）
- 发票详情页（含打印预览）
- 发票编辑（仅草稿状态）
- 发票状态管理（草稿→已开票→已收款/逾期/已作废/已退款）
- 自动生成发票号（INV-YYYYMMDD-序号格式）
- 发票打印功能
- 购方税号和开票地址管理
- 金额自动计算（小计、折扣、税额、价税合计）

### 10. 产品管理
- 产品列表（支持搜索和分类筛选）
- 新建/编辑/删除产品
- 产品详情页
- 产品分类管理（软件、硬件、服务、咨询、其他）
- SKU自动生成
- 成本价和利润率计算
- 产品启用/禁用状态管理
- 支持报价单和订单选品

### 11. 回款管理 (V3.3 新增)
- 回款计划列表（支持搜索和状态筛选）
- 统计卡片（应收总额、已收金额、逾期金额、回款率）
- 新建回款计划（关联客户/商机/合同）
- 回款进度跟踪（部分回款/全额回款）
- 逾期自动计算和标记
- 回款登记（支持多种收款方式：银行转账、现金、信用卡等）
- 标签页快速筛选（全部/逾期/待收款/已结清）

### 12. 系统设置
- 通知设置
- 数据管理选项
- 快捷键提示

## 技术栈

- **框架**: Next.js 16 (App Router)
- **UI组件**: shadcn/ui (基于 Radix UI)
- **样式**: Tailwind CSS 4
- **语言**: TypeScript 5
- **状态管理**: React Context + useState
- **数据库**: Supabase

## 快速开始

### 安装依赖

```bash
pnpm install
```

### 启动开发服务器

```bash
pnpm dev
# 或者使用 coze dev
```

启动后，在浏览器中打开 http://localhost:5000 查看应用。

### 构建生产版本

```bash
pnpm build
# 或者使用 coze build
```

### 启动生产服务器

```bash
pnpm start
# 或者使用 coze start
```

## 项目结构

```
src/
├── app/
│ ├── api/
│ │ └── search/       # 全局搜索 API
│ ├── customers/      # 客户管理模块
│ ├── contacts/       # 联系人管理模块
│ ├── opportunities/  # 销售机会管理模块
│ ├── quotes/         # 报价单管理模块
│ ├── contracts/      # 合同管理模块
│ ├── orders/         # 订单管理模块
│ ├── invoices/       # 发票管理模块
│ ├── products/       # 产品管理模块
│ ├── settings/       # 系统设置
│ └── page.tsx        # 仪表盘
├── components/
│ ├── crm/
│ │ ├── global-search.tsx  # 全局搜索组件
│ │ ├── sidebar.tsx        # 侧边栏
│ │ ├── header.tsx         # 顶部栏
│ │ └── layout.tsx         # CRM布局
│ └── ui/             # shadcn/ui组件
└── lib/
 ├── crm-context.tsx   # CRM状态管理
 ├── crm-database.ts   # 数据库服务
 └── crm-types.ts     # 类型定义
```

## 数据说明

数据存储在 Supabase 数据库中，支持持久化存储。

## 响应式设计

系统支持PC端和移动端自适应：

- **PC端**: 使用侧边栏导航 + 表格展示数据
- **移动端**: 使用汉堡菜单 + 卡片展示数据
