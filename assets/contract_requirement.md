# CRM合同管理模块需求

## 一、功能概述
基于已有的产品报价模块，延伸开发合同管理功能，完成"报价→合同→回款"商业闭环。

## 二、核心功能点

### 1. 合同列表页
- 显示所有合同，支持筛选（客户、状态、日期范围）
- 列表展示：合同编号、客户名称、金额、签订日期、状态
- 支持新增、编辑、删除、详情查看

### 2. 合同详情页
- 基本信息：合同编号、客户、签订日期、有效期
- 产品明细：从报价单关联或手动添加
- 金额信息：合同总金额、已回款金额、待回款金额
- 状态管理：草稿→执行中→已完成/已终止

### 3. 合同创建/编辑
- 选择关联客户（从已有客户列表选择）
- 添加产品明细（可关联报价单）
- 设置金额、日期、有效期
- 合同条款（富文本）

### 4. 合同状态管理
- 草稿：可编辑
- 执行中：正常履行
- 已完成：全部回款
- 已终止：提前结束

### 5. 合同关联
- 关联客户档案
- 关联产品
- 关联回款记录

## 三、技术要求
- 使用现有数据模型扩展
- 复用现有UI风格（card-elevated、section-title等）
- 添加合同管理菜单到侧边栏
- 保持与其他模块风格一致

## 四、验收标准
- [ ] 合同列表页正常显示和筛选
- [ ] 可以创建、编辑、删除合同
- [ ] 合同详情页完整展示所有信息
- [ ] 状态流转正常
- [ ] UI风格与现有模块一致
