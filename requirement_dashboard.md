# CRM仪表盘增强功能开发需求

## 项目背景
CRM项目地址：https://github.com/Frank-zhao-junjun/CRM
部署域名：https://vfdm2xmfj2.coze.site
技术栈：Next.js 16 + TypeScript + TailwindCSS + Shadcn/UI + Prisma

## 功能需求：仪表盘增强

### 1. 新增统计卡片
在现有仪表盘基础上，新增以下统计指标：
- **本月销售额**：统计当月所有已完成订单的总金额
- **本月订单数**：统计当月新增订单数量
- **本月新增客户数**：统计当月新增客户数量
- **本月报价单数量**：统计当月新增报价单数量

### 2. 新增销售趋势图
- **月度销售额趋势**：折线图展示最近6个月的销售额变化
- **月度订单量趋势**：折线图展示最近6个月的订单数量变化

### 3. 订单状态分布饼图增强
- 点击饼图扇区可穿透查看对应状态的具体订单列表
- 显示各状态的数量和占比

### 4. TOP排行榜
- **TOP 5 客户**：按订单金额排序，显示客户名称和金额
- **TOP 5 产品**：按销量排序，显示产品名称和销售数量

### 5. 销售漏斗增强
- 展示销售机会各阶段的数量
- 计算阶段转化率

## 设计要求
- 统计卡片使用统一的卡片样式，带图标
- 图表使用ECharts或Recharts
- 保持与现有仪表盘风格一致
- PC端适配

## 数据库说明
- 订单(Order)：amount字段存储金额，status存储状态
- 客户(Customer)
- 报价单(Quote)
- 销售机会(Opportunity)
