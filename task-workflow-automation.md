# 工作流自动化引擎 - V5.2

## 需求背景
当前CRM已有销售自动化序列（拖拽式序列编辑器），但缺少通用工作流引擎。需要构建一个可视化的工作流自动化系统，支持触发器+条件的配置，以及多种动作执行。

## 核心功能

### 1. 工作流列表与统计
- 工作流列表页面（搜索、状态筛选）
- 统计卡片：总工作流数、活跃数、执行成功率、今日执行数
- 创建/编辑/删除/启用/禁用工作流

### 2. 工作流编辑器
- 可视化拖拽式编辑器
- 节点类型：
  - **触发器**：定时触发（cron）、事件触发（客户创建/更新、商机阶段变化、回款到账等）
  - **条件**：IF/ELSE 条件分支
  - **动作**：发送邮件、创建任务、发送提醒、更新字段、发送Webhook

### 3. 工作流执行引擎
- 后台定时任务扫描触发器
- 事件监听器处理实时触发
- 执行日志记录与追踪

### 4. 预置模板
- 新客户欢迎邮件
- 商机阶段变更通知
- 回款逾期提醒
- 线索X天未跟进提醒

## 技术要求
- 继续使用 Next.js App Router + shadcn/ui
- Supabase 存储工作流配置和执行日志
- 使用 React Flow 或类似库实现可视化编辑器
- API 路由处理工作流 CRUD 和执行
